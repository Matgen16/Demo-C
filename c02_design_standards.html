Web Design Standards

Principles of Web Design - CRAP

  1. Contrast (make it easy to understand structure and content at a glance)
  2. Repetition (establish a style and be consistent throughout)
  3. Alignment (position elements to flow smoothly from one to the next)
  4. Proximity (cluster elements so they fit together in the way they're consumed naturally)

Universal Design Principles - POUR

  1. Perceivable (can all content be accessed by any user?)
  2. Operable (can anyone navigate and interact with all features?)
  3. Understandable (is layout confusing or is content presented inconsistently?)
  4. Robust (is it designed to fail gracefully? Is HTML flow logical? Does the code adhere to standards?)
