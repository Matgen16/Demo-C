Wireframing and Web Design

A wireframe is a boxy sketch showing the intended layout of each Web page. Lorem Ipsum is commonly used for placeholder text when designing so that you can see whether your content will be crowded or difficult to read even before you have actual content to add.

Code editors (like VS Code) and Integrated Development Environments (like IntelliJ IDEA) frequently have ways of quickly inserting Lorem Ipsum elements as needed. If you are going to mock up a Website by directly coding it, you should read up on how to make this work for you.

Ideally, you want to include your color palette in your wireframe. Once we learn how to code CSS Grid, this will be an easy thing to do. For now, however, it will take some patience on your part while you are using tools such as Google Drawing to wireframe your pages.
