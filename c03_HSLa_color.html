HSLα is the Best Way to Control Website Colors

Although hexadecimal remains the most common way to code colors, it is an old method that should be discontinued. You will not be permitted to use hexadecimal colors in this class, so do not spend time learning how. Instead, I will require you to choose between two different methods:

  • HTML color names
  • HSLα color codes

Of the two methods, HSLα is far superior in general, but you should feel free to use plain color names for the sake of being quick, especially when coding deliberate Crayola-type colors like black and white. You will also see some developers using RGBα codes. Those work as well as HSLα but they were designed around the way machines process color rather than around the way that human brains process color. HSLα is designed to be precise enough for machines but written in a way that is human-readable if you learn the color wheel and remember that there are 360Â° in a circle.

Using HSLα: Hue

These are the hue values for the basic rainbow hues:

  1.   0 → red
  2.  30 → orange
  3.  60 → yellow
  4. 120 → green
  5. 240 → blue
  6. 270 → purple

Using HSLα: Saturation

Saturation is a percentage showing how much of the color you want to add. 100% saturation is a pure, jewel-tone version of the color. 0% saturation would have no color present at all, and this is the default if you are using HSLα to code a shade of gray. Your default when using color, however, should be to start at 100% saturation and then tone it down to reach the level of color intensity you desire. Note that low saturation is analogous to low contrast, so be cautious if you need to violate that particular design principle.

Using HSLα: Luminance

Luminance is a percentage showing how much light you want to shine through your color. You should default to 50% luminance because 0% would turn any color to black and 100% would turn any color to white; 50% luminance is just the pure hue and saturation you chose, without washing it out or shading it down.

Using HSLα: Alpha

Alpha (&alpha;) is a measure of opacity in an image or a color. Although it might make more sense to set it as a percentage the same way you would set saturation or luminance, the standard is to set it as a value between 0.00 and 1, where 0 would be completely transparent and 1 would be completely opaque. You can achieve wonderful subtleties with your color palette when you layer elements and tweak alpha values, and I encourage you to play around with this part of the HSLα code.

Using Adobe Color to Select a Color Palette

I encourage you to go over to Adobe Color and use their online tools to refine your Website aesthetics. Once we get into using Cascading Style Sheets (which is our next unit) you'll be able to start implementing all of these splashy colors you've been exploring.

These are the common color schemes you can use to guide your selections:

  • monochromatic - pick one color as an accent and leave everything else black and white
  • analogous - adjacent colors on the color wheel - mimic color schemes found in nature
  • complimentary - opposite colors on the color wheel - bad for text
  • split complimentary - adds one adjacent color to soften the effect of the complimentary scheme
  • triadic - 3 colors evenly spaced around the color wheel - good for vibrancy
  • tetradic - 4 colors comprised of 2 complimentary pairs
